Quick start

You can grab the source and build the project from scratch if you have gcc 4.7.2 (or higher) and libevent2:

$ git clone git://github.com/victorparmar/zsearch.git
$ cd zsearch
$ ./buildall.sh
$ ./build/server ./docroot
